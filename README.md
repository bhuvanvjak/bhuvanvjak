### 📊 EPM Financial Summary Agent
**Local LLM-Powered Finance Automation**
```
🔧 Tech Stack: React • SheetJS • Mistral 7B (LM Studio) • Local LLM
🎯 Purpose: Auto-generate monthly revenue variance commentary from Excel Flash 
            Decks — fully on-device, no data ever leaves the machine
⭐ Highlights: 2-3 hour manual task reduced to a 1-click, sub-minute generation; 
              surgical 50-cell extraction (~400 tokens) eliminates hallucinated numbers; 
              supports both WD5 (Preliminary) and WD7 (Final) close cycles
```
